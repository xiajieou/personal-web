# xia jie ou — portfolio

Personal site built with **Next.js 13**, **Chakra UI**, and **Framer Motion**.
Dark-first, techy, mobile-friendly, optimized for Lighthouse.

## scripts

```bash
npm run dev     # http://localhost:3000
npm run build
npm run start
npm run lint
```

## stack

- **framework:** Next.js 13 (pages router)
- **ui:** Chakra UI + custom theme
- **fonts:** Inter + JetBrains Mono via `next/font`
- **motion:** Framer Motion
- **analytics:** @vercel/analytics

## structure

```
components/   # hero, navbar, project-card, etc.
pages/        # index.js is the whole site
lib/theme.js  # chakra theme + global styles
public/       # favicons + profile image
```
